# Changelog

All notable changes to `servecore-shadcn-blade` will be documented in this file.

## [1.0.0] - 2026-01-21

### Added
- Initial release of ServeCore Shadcn Blade preset
- Automatic installation command (`php artisan servecore:install`)
- Theme system with Dark/Light/System modes
- Flicker-free theme initialization
- Pre-built components:
  - Button (with loading state)
  - Dialog (with nested button fix)
  - Select (with Alpine.js integration)
  - Tabs (with Alpine.js state management)
  - Progress (pure CSS implementation)
  - Theme Toggle (cycle button)
  - Theme Script (FOUC prevention)
- Tailwind CSS v4 configuration with ShadCN color variables
- Alpine.js integration for component reactivity
- Automatic NPM dependency installation (embla-carousel)
- Comprehensive documentation and usage examples

### Features
- Zero-config installation
- Self-contained components (no external JS registration needed)
- Developer-friendly architecture
- Full dark mode support
- Accessible UI components following ShadCN principles
